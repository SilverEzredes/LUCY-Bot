# L.U.C.Y. Bot
Custom Discord Bot for the Uncharted/The Last of Us Modding Servers.

## Features:
 - Leveling system for:
   - Level
   - Cred
   - Assistance
 - Reputation command
 - Daily reward
 - Mod requests system
 - Dice command
 - CyberCookie and SpaceBurrito commands
 - Level up notification
 - Join message
